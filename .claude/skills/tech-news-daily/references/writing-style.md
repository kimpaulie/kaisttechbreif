# 글쓰기 룰 (F4)

SKILL.md `[6] 작성` 단계의 세부 룰. 모든 텍스트 필드(`title`, `summary`)에 적용.

## 1. 제목 (`title`)

| 항목 | 규칙 |
|---|---|
| 길이 | **24~29자** (공백·구두점·이모지 포함). 30자 넘기지 말 것. 사이트 카드 한 줄 안에 들어와야 함 |
| 형식 | "주체, 사건/숫자" 또는 "주체 사건 — 숫자" |
| 필수 포함 | 회사명·기관명 등 **고유명사 1개 이상** + **핵심 숫자/규모** |
| 톤 | 객관적, 사실 중심. 평가어 회피 |

### 길이 감각 (좋은 예)

| 글자 수 | 제목 |
|---|---|
| 27자 | "한국 보안업계, 내수에서 중동·동남아 진출 본격화" |
| 29자 | "앤트로픽, 밀라노 사무소 개설… 유럽 매출 9배 성장" |
| 28자 | "엔비디아, 다음 분기 매출 125조 원 가이던스" |

### 줄이는 기술 (30자 초과 시)

1. **부제·꼬리 표현 삭제**: `"… 사상 최대"`, `"… 전망"` 같은 꼬리표 제거
2. **수식어 압축**: `사상 최대 → 사상 첫`, `대규모 → 대형`
3. **연결어 압축**: `~로 → ~`, `~에서 → 생략`
4. **숫자 단위 줄이기**: `1,170조 원 → 1,170조`, `약 125조 원 → 125조`
5. **회사명 약식**: `삼성전자 → 삼성`, `SK하이닉스 → SK` (단, 헷갈리지 않는 맥락에서만)

### 좋은 예
- ✅ "Nvidia, 사상 첫 시가총액 5조 달러 돌파"
- ✅ "OpenAI, 한국에 30억 달러 첫 아시아 데이터센터"
- ✅ "삼성전자, 2nm 파운드리 양산 일정 6개월 앞당겨"

### 나쁜 예
- ❌ "AI 회사가 큰 투자를 받았다" (구체성 0)
- ❌ "충격! 시총 5조 달러 돌파!" (자극어 + 느낌표)
- ❌ "엔비디아의 무서운 성장" (주관어)

## 2. 본문 요약 (`summary`)

### 정확히 3문장 산문

| 문장 | 역할 | 패턴 |
|---|---|---|
| **1문장** | What — 핵심 사실 | "[주체]가 [행위/결과]하였습니다." |
| **2문장** | Why/How — 배경·세부·숫자 | "[배경 또는 디테일]입니다." |
| **3문장** | So What — 의미·한국 함의 | "[국내 산업/기업에 미치는 영향]으로 전망됩니다." |

### 읽기 난이도 (중요)

**중학생도 한 번 읽고 이해할 수 있어야 한다.** MBA 페르소나는 깊이 있는 통찰을 원하지만, 문장 자체는 평이해야 정보가 빨리 들어온다.

#### 쉽게 쓰는 7가지 룰

1. **한 문장 60자 안팎 (75자 한도)** — 가장 중요한 룰. 60자 넘으면 끊는 걸 우선 검토하고, 75자 넘기면 무조건 두 문장으로 분리. 한 문장은 한 호흡에 읽혀야 한다.
2. **종속절은 한 문장에 최대 1개**, 가능하면 0개. "~하는 가운데 ~하면서 ~하기 위해" 같은 다중 종속 절대 금지.
3. **전문 약어는 풀어쓰기**: `HBM4` → `메모리 칩(HBM4)`, `LLM` → `대형 언어모델(LLM)`, `EUV` → `노광 장비(EUV)`. 처음 한 번 풀어주고 두 번째부터 약어만.
4. **한자어 → 일상어 대치** (강화 표):

   | 한자어/문어 | 일상어 |
   |---|---|
   | 재편 | 다시 짜기 / 구조 바꾸기 |
   | 가속화 | 빨라지다 |
   | 본격화 | 본격 시작 |
   | 시사 | 암시 / 보여주다 |
   | 우호적 | 유리한 |
   | 도달 | 닿다 / 이르다 |
   | 확보 | 따내다 / 손에 넣다 |
   | 진출 | 들어가다 |
   | 추진 | 밀다 |
   | 전망 | 보일 것 / 예상 |
   | 해석 | 풀이 / 받아들임 |
   | 야기 | 일으키다 |
   | 도모 | 꾀하다 / 노리다 |
   | 부각 | 떠오르다 |

   단, 고유명사(회사·정책·기술 이름)는 원문 유지.

5. **수동태 줄이기**: `~되었다` → `~했다`, `~확인된다` → `~확인했다`. 단, 인용 출처를 가릴 때는 수동태 OK.
6. **고유명사·전문 개념은 미니 설명 한 줄**: `루빈(엔비디아 차세대 GPU)`, `HBM4(고대역폭 메모리 4세대)`.
7. **수식어 가지치기**: "~의 ~한 ~인" 같은 관형 수식이 두 번 이상 겹치면 한 번은 자른다.

#### 자체 검수: 한 호흡 룰

각 문장을 **속으로 끝까지 읽어 봤을 때 중간에 끊어 다시 시작하고 싶은 충동**이 들면 그 문장은 너무 길거나 복잡한 것이다. 끊고 단순화한다.

#### Before / After 예시 (5/26 호 실제 문장 압축)

**원칙**: 3문장 룰은 그대로 유지. 단, **각 문장을 60자 안에 들어가도록 압축**한다. 정보가 많으면 덜 중요한 디테일을 과감히 삭제.

**예시 1**

❌ **나쁨** (95자, 종속절 3개):
> 분기 매출 100조 원대 가이던스는 엔비디아에 메모리(HBM)를 공급하는 SK하이닉스와 삼성전자에 그대로 추가 주문으로 흘러갈 것으로 전망됩니다.

✅ **좋음** (50자):
> 분기 100조 원대 가이던스는 한국 HBM 공급망(SK·삼성)에 그대로 추가 주문으로 직결됩니다.

**예시 2**

❌ **나쁨** (95자, 두 사실 묶음 + 종속절 다중):
> 핵심은 트랜지스터를 더 작게 그리는 대신 칩 안에서 부품을 입체적으로 쌓고 배선을 짧게 만들어 같은 면적에 53.5% 더 많은 트랜지스터를 넣는 방식이라고 허팅보 하이실리콘 사장이 설명하였습니다.

✅ **좋음** (58자):
> 허팅보 하이실리콘 사장은 부품을 입체로 쌓는 방식으로 같은 면적에 53.5% 더 많은 트랜지스터를 넣었다고 설명했습니다.

**예시 3**

❌ **나쁨** (90자):
> 두 회사를 합친 한국 메모리 영업이익은 올해 사상 최대인 204조 원에 이를 전망이며, 이번 지명은 단순 부품 계약을 넘어 HBM 시장이 양강 구도로 바뀌는 신호로 해석됩니다.

✅ **좋음** (55자):
> 한국 메모리 영업이익은 올해 204조 원이 예상되며, 이번 지명은 HBM 시장이 양강 구도로 바뀌는 첫 신호입니다.

> **꼭 기억**: 정보를 다 욱여넣지 말고, **중요한 한 가지**만 한 문장에 담는다. 빠진 디테일은 출처 링크가 알아서 보충한다.

### 종결어미 (존댓말)

- "~합니다", "~하였습니다", "~밝혔습니다", "~전망됩니다", "~예상됩니다", "~분석됩니다"
- 평어체("~다"), 반말, 명사 종결("~함") 모두 금지.
- 친절한 톤이되 늘어지지 않게: `~하였습니다`가 너무 잦으면 `~했습니다`로 줄여도 OK (의미 동일, 더 가벼움).

### 형식 제약

- **글머리표 절대 금지**. `•`, `-`, `*`, 번호 매기기 등 일체 사용하지 않는다.
- 줄 바꿈도 사용하지 않는다. 한 문단의 연속된 산문이어야 한다.
- 따옴표 안에 인용 가능 (예: 팀 쿡 CEO가 "공간 컴퓨팅 대중화의 원년"이라고 언급하였습니다).

## 3. 숫자 표기

| 종류 | 규칙 | 예시 |
|---|---|---|
| 큰 달러 액수 | **반드시 원화 환산 병기** | "900억 달러(약 124조 원)" |
| 큰 한국 원화 | 단위 약식 (조/억) | "1조 2,000억 원" |
| 비율 | %, 소수점은 1자리까지 | "4.2% 급등" |
| 비교 단위 | 한국인 체감 가능한 비교 추가 가능 | "삼성전자 시총과 비슷한 규모" |
| 시점 | 명확히 (분기·년) | "2026년 4분기" |

## 4. 금지어 (절대 사용 금지)

### 자극어
충격, 역대급, 대박, 미쳤다, 엄청난, 굉장한, 어마어마한, 폭풍, 폭격, 핵폭탄

### AI 정형어
결론적으로, 요컨대, 다음과 같이, ~라고 합니다, 종합하면, 정리하면, 한마디로

### 모호어
"많은", "큰", "다양한" 같은 형용사는 가능한 한 구체적 숫자/사례로 대체.

## 5. 인용·따옴표

- 임원 발언 인용은 직접화법으로 자연스럽게 녹인다.
- 예: `샘 올트먼 CEO는 "한국이 아시아 AI 허브의 중심"이라고 강조하였습니다.`

## 6. 출처명(`sources[].name`) 표기

- 한국어 매체: 풀 네임 그대로. 예: `"한국경제"`, `"ZDNet Korea"`, `"매일경제"`.
- 영문 매체: 정식 영문 명칭. 예: `"Financial Times"`, `"The Wall Street Journal"`, `"Bloomberg"`.
- 약어 사용 금지. ✅ `"The Wall Street Journal"`, ❌ `"WSJ"`.

## 7. URL(`sources[].url`)

- `https://` 프로토콜 필수.
- 가능한 한 **해당 기사**의 직접 URL. 메인 페이지로 떨어지지 않게.
- 트래킹 파라미터(`?utm_...`)는 제거.
